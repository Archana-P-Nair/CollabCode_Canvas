## 🏁 Running Locally

### 1. Clone repo

```bash
git clone https://github.com/Archana-P-Nair/CollabCode_Canvas.git
cd CollabCode_Canvas
```

### 2. Install dependencies

**Frontend**

```bash
cd frontend
npm install
```

**Backend**

```bash
cd ../backend
npm install
```

### 3. Run backend

```bash
cd backend
npm run dev
```

### 4. Run frontend

```bash
cd frontend
npm run dev
```

Then open:

```text
http://localhost:3000
```
